Знакомство с компилятором и первая программа

Компилятор - это программа, которая превращает код на человекочитаемом языке в машинный код.

У C++ несколько компиляторов. Самый известный это Gnu Compiler Collection (GCC). Его и будем использовать.

Чтобы скомпилировать код на C++ в Unix операционке нужно написать:

g++ file.cpp -o file

Чтобы запустить:

./file

У языка C++ много различных стандартов. Чтобы использовать какой-то определенный при компиляции, надо написать:

g++ --std=c++17 file.cpp -o file

Также в комиляторе g++ существует встроенный оптимизатор трех различных видов (-O1, -O2, -O3). Чтобы скомпилироватьс оптимизацией нужно написать:

g++ -O2 file.cpp -o file

Еще можно указать -Wall, чтобы компилятор предупреждал о максимальном количестве проблем в коде:

g++ -Wall file.cpp -o file

Если указать -Werror, то компилятор будет считать все warning'и ошибками:

g++ -Werror -Wall file.cpp -o file

Ну и -Wextra, чтобы компилятор проверял каждый чих (все равно не спасет от ошибок :))

g++ -Werror -Wall -Wextra file.cpp -o file

Код первой программы:

#include <iostream> // input output stream (ввод и вывод)

int main()
{
    int x; // объявление переменной (целое число)
    std::cin >> x; // считывание из потока ввода в переменную
    std::cout << x << "\n"; // запись в поток вывода
}
